# Talk with Hisham

A personal portfolio and social platform for real-time debates and communication by **Muhibbullah Hisham**.

## Features

- 🏠 **Landing Page** — Stunning hero section with animated particles and glassmorphism
- 📝 **Public Feed** — Share opinions and engage with posts (like, share)
- 💬 **Live Chat** — Real-time chat room for public debate and discussion
- 👤 **Profile System** — User profiles with avatar upload and bio
- 🔐 **Authentication** — Sign in/sign up with email via Supabase Auth
- 📱 **Responsive** — Full mobile support with hamburger menu

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · Supabase · Framer Motion · ShadCN UI

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. (Optional) Set up Supabase for full functionality:
   - Create a project at [supabase.com](https://supabase.com)
   - Run `supabase_setup.sql` in the SQL Editor
   - Copy `.env.example` to `.env.local` and fill in your keys

3. Run the app:
   ```bash
   npm run dev
   ```

> **Note:** The app works in demo mode without Supabase — Feed and Chat use local state, and Profile shows a default view.
